微信小程序

发表于 2018-06-06

主要文件作用：

  1. app.json文件：是对当前小程序的全局配置，包括小程序的所有页面路径（pages:[]）、window界面表现（window:{}）、底部tab（tabBar:{}）、设置网络超时时间（networkTimeout:{}）等配置
    1. pages：数组的第一项代表小程序的初始页面。小程序中增加或者减少页面都需要对pages数组进行修改
    2. window：用于设置小程序的状态栏、导航条、标题、窗口背景色。注意：backgroundColor 是下拉窗口的背景色不是整个页面的背景色，整个页面背景色可以在app.wxss里面设置page{background:’#fff’}

发表于 2018-05-29

前端页面由哪三层组成：

结构层：HTML，表现层：CSS，行为层：JavaScript；

如何利用边框写出三角形：

  1. 将div的宽高都设为0

  2. 需要哪个方向是三角形就把哪个方向的边框颜色改变即可

  3. 把其他方向的边框颜色变成transparent

  4. 去掉对边边框即可完成

    div{width: 0px; height: 0px; border: 200px solid transparent; border-bottom-color: yellow; border-top: 0;}；

vue

vue-cli

安装步骤

  1. 全局安装vue-cli：npm install –global vue-cli
  2. 创建一个基于webpack模板的新项目：vue init webpack my-project
  3. 进入项目：cd my-project
  4. 安装依赖：npm install
  5. 运行：npm run dev
  6. 打包：npm run build 生成dist文件夹

主要文件作用

  1. index.html——展示页面
  2. index.js——引入一些配置文件 （import from 同级目录用./，@就是../）
  3. App.vue——根组件，展示页面得结构，引样式表用@import ‘目录’
  4. components——子组件 export default{}
  5. 所有得静态资源全部放在static文件夹下

git和github

git和github的区别

git是版本控制工具

github是网站 ：社交平台、开源项目、远程仓库

svn和git的区别

svn为集成式，要进行版本控制必须联网，要经过中央服务器才能进行版本控制，非常依赖于中央服务器

git为分布式，不用网络就可以进行版本控制，在本机进行版本控制就好

使用hexo框架搭建博客的步骤

发表于 2018-05-23

使用Hexo搭建博客的步骤：

创建基于hexo的项目

  1. npm install hexo-cli -g 安装
  2. hexo init blog（项目名称） 初始化项目
  3. cd blog 进入项目
  4. npm install 下载模块
  5. hexo server

配置主题：NexT主题安装步骤

  1. 在blog项目中 git clone https://github.com/iissnan/hexo-theme-next themes/next
  2. 启用主题：打开站点配置文件_config.yml，找到theme字段，并将其值改为next
  3. 验证主题是否正确启用，先用hexo.clean来清缓存，再用 hexo s –debug 启动
