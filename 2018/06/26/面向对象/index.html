面向对象

发表于 2018-06-26

定义：面向对象是一种编程思想，就是把一些具有相同特征的归为一类，并且返回一个对象，这个对象的特性是挂在构造函数的原型上面的。

特征：

  1. 抽象：抽出像的部分
  2. 封装：封函数，只能通过对象来访问方法
  3. 继承

构造函数：构造对象的函数，一个函数return出一个对象，那么这个函数就是构造函数

实例化对象：把构造函数return的对象用一个变量存起来，那么这个变量就是实例化对象

属性和方法：

  1. 前面无主的叫变量 var n = 0;，前面有主的叫属性 div.index
  2. 前面无主的叫函数 function fn(){}，前面有主的叫方法 array.push
  3. 只有对象（包含内置对象）下面才有属性和方法，简单类型没有

new：

  1. new是一个运算符，专门用来运算函数
  2. 使用了new之后：
    1. 如果不写括号，不传参就等同于函数的调用
    2. 默认返回值不是undefined，而是obj，如果写了return并且return出来的是对象类型，那么返回值就为return出来的那个对象，如果没写return，返回值就是new出来的那个对象
    3. 构造函数中的this指向new出来的那个对象
    4. 内置对象的方法是不能new的，只能new自己写的方法

原型（prototype）：

  1. 改写对象下面的属性和方法，让共用的属性或者方法在内存中只存一份（目的是为了解决实例化对象使用方法时性能的问题，提高性能）
  2. 每个函数都自带一个属性或者方法，其中有一个属性：prototype
  3. prototype的值是一个对象
  4. prototype只能给该构造函数的实例化对象使用
  5. js有一个特性，如果实例化对象上没有某个属性或者方法，那么他一定会去构造函数的原型上找这个属性或者方法
  6. 构造函数：属性写在构造函数里，方法写在构造函数的原型上

原型链：

  1. 对象身上一定有原型链，一定没有原型
  2. 原型链是构造函数原型与实例化对象之间的桥梁
  3. 实例化对象的原型链等于构造函数的原型
