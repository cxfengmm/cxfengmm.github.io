发表于 2018-05-29

前端页面由哪三层组成：

结构层：HTML，表现层：CSS，行为层：JavaScript；

如何利用边框写出三角形：

  1. 将div的宽高都设为0

  2. 需要哪个方向是三角形就把哪个方向的边框颜色改变即可

  3. 把其他方向的边框颜色变成transparent

  4. 去掉对边边框即可完成

    div{width: 0px; height: 0px; border: 200px solid transparent; border-bottom-color: yellow; border-top: 0;}；

自动显示省略号：

单行自动显示省略号：

  1. width:200px;
  2. white-space:nowrap;
  3. overflow:hidden;
  4. text-overflow:ellipsis;

多行自动显示省略号：

  1. overflow: hidden;
  2. text-overflow:ellipsis;
  3. display: -webkit-box;
  4. -webkit-line-clamp:2; （两行出省略号）
  5. -webkit-box-orient:vertical;

CSS引入方法有哪些：

  1. 行间样式
  2. 内部样式
  3. 外部样式

link和@import的区别：

  1. 老祖宗的区别：link属于XHTML标签，除了可以加载css样式表外还能定义RSS和rel连接属性等，而@import只能加载css样式
  2. 加载顺序的区别：页面加载时，link引用的css样式表会同时加载，而@import引用的css样式会等到页面加载完成之后再加载
  3. 兼容性的区别：@import是CSS2.1提出的，只能兼容ie5以上的浏览器，而link没有兼容性问题
  4. 使用DOM控制样式时的区别：当使用JS控制DOM去改变样式时，只能用link，因为@import是不被DOM控制的

前端开发：

  1. 所有代码的书写必须是英文半角状态下的小写
  2. 使用tab键锁进
  3. 所有的标签必须闭合
  4. 内联元素不能包块元素
  5. a标签不能嵌套a标签
  6. 结构，表现，行为分离

margin-top给父元素的第一个子元素写会出现什么问题，如何解决：

问题：margin-top写给第一个子元素会附加在父元素身上

解决办法：

  1. 给父元素加border
  2. 给父元素加overflow:hidden;
  3. 处理这类问题时尽量用padding-top代替

margin-bottom同理

padding不能取负值

样式重置表：

  1. body{background:#fff;}
  2. body,p,h1~h6{margin:0;}
  3. ul,ol{margin:0;padding:0;}
