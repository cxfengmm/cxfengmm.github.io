发表于 2018-05-29

前端页面由哪三层组成：

结构层：HTML，表现层：CSS，行为层：JavaScript；

如何利用边框写出三角形：

  1. 将div的宽高都设为0

  2. 需要哪个方向是三角形就把哪个方向的边框颜色改变即可

  3. 把其他方向的边框颜色变成transparent

  4. 去掉对边边框即可完成

    div{width: 0px; height: 0px; border: 200px solid transparent; border-bottom-color: yellow; border-top: 0;}；

自动显示省略号：

单行自动显示省略号：

  1. width:200px;
  2. white-space:nowrap;
  3. overflow:hidden;
  4. text-overflow:ellipsis;

多行自动显示省略号：

  1. overflow: hidden;
  2. text-overflow:ellipsis;
  3. display: -webkit-box;
  4. -webkit-line-clamp:2; （两行出省略号）
  5. -webkit-box-orient:vertical;

CSS引入方法有哪些：

  1. 行间样式
  2. 内部样式
  3. 外部样式

link和@import的区别：

  1. 老祖宗的区别：link属于XHTML标签，除了可以加载css样式表外还能定义RSS和rel连接属性等，而@import只能加载css样式
  2. 加载顺序的区别：页面加载时，link引用的css样式表会同时加载，而@import引用的css样式会等到页面加载完成之后再加载
  3. 兼容性的区别：@import是CSS2.1提出的，只能兼容ie5以上的浏览器，而link没有兼容性问题
  4. 使用DOM控制样式时的区别：当使用JS控制DOM去改变样式时，只能用link，因为@import是不被DOM控制的

前端开发：

  1. 所有代码的书写必须是英文半角状态下的小写
  2. 使用tab键锁进
  3. 所有的标签必须闭合
  4. 内联元素不能包块元素
  5. a标签不能嵌套a标签
  6. 结构，表现，行为分离

margin-top给父元素的第一个子元素写会出现什么问题，如何解决：

问题：margin-top写给第一个子元素会附加在父元素身上

解决办法：

  1. 给父元素加border
  2. 给父元素加overflow:hidden;
  3. 处理这类问题时尽量用padding-top代替

margin-bottom同理

padding不能取负值

样式重置表：

  1. body{background:#fff;}
  2. body,p,h1~h6{margin:0;}
  3. ul,ol{margin:0;padding:0;list-style:none;}
  4. strong,b,h1~h6{font-weight:normal;}
  5. em,i{font-style:normal;}
  6. a{color:#000;text-decoration:none;}
  7. img{border:0;vertical-align:top;}
  8. table{margin:0;padding:0;}
  9. input,select,option,textarea,fieldset{margin:0;padding:0;background:none;border:0;outline:none;vertical-align:top;}

strong和b标签有啥区别，em和i标签有啥区别：

strong和b标签都可以加粗，都默认有加粗样式

strong是结构化标签，不仅可以加粗，还具有语义化表示强调，b是表现化标签，不能表示强调

em和i标签都默认有倾斜样式

em是结构化标签，不仅可以倾斜，还具有语义化表示强调，i是表现化标签，不能表示强调

行内元素有哪些？块级元素有哪些？空元素有哪些？

  1. 行内元素：span，a，strong，b，em，i，input，button，select，textarea，img
  2. 块级元素：div，p，h1~h6，dl，dt，dd，ol，ul，li
  3. 空元素：br，hr，link，img，input，meta

CSS盒模型：

正常的盒模型是content+padding+border+margin

ie的盒模型是将padding和border往里算的

内联元素与块级元素有啥区别，如何转换：

内联标签：

  1. 写不上宽高，宽高由内容决定
  2. 默认文档流是左右排列
  3. 不支持上下margin
  4. 对于padding是支持左右的，对于上下的padding位置不能正常显示
  5. 代码换行会被解析成一个空格

块标签：

  1. 宽默认占满一行
  2. 默认文档流是上下排列
  3. 支持所有的css样式

转换：display:inline; ——将块元素转成内联元素

​ display:block; ——将内联元素转成块元素

CSS选择符有哪些？优先级排列，!important和内联元素哪个优先级高？

  1. 选择符：标签选择符（p），类选择符（class），id选择符（#），包含选择符（div p），父子选择符（div>p），内联选择符（a），群组选择符，通配符，伪类选择符
  2. 优先级：内联选择符1000>id选择符0100>类选择符0010>标签选择符0001；包含选择符权重是所有选择符之和，群组选择符是他们各自本身，伪类选择符是0010加上前面选择符的权重
  3. !important优先级高于内联元素的优先级

哪些属性可以继承：

文本属性和列表属性可以继承

五大浏览器内核及其代表：

  1. Trident：ie,360浏览器
  2. Gecko：火狐浏览器
  3. Webkit：Safari浏览器
  4. Presto：Opera浏览器
  5. Blink

img标签的title和alt属性区别是什么：

title：鼠标悬停在上面显示的提示信息

alt：当图片无法正常加载时出现在图片区域的替代文本

浮动过后对元素的影响：

浮动过后的元素既有块元素的特点又有内联元素的特点。所有元素都在一排左右显示，脱离文档流

清浮动的方法：

  1. 空标签清浮动：在塌陷的元素里添加一个块元素作为他的最后一个子元素并写上样式：（height:0;overflow:hidden;clear:both;）
  2. br清浮动（不常用，不规范）
  3. 伪类清浮动：（.clearfix:after{ content:””; display:block; clear:both; height:0; overflow:hidden; visibility:hidden;} .clearfix{ zoom:1;} ）
  4. 给父元素定高

BFC：

作用：

  1. 包含浮动元素
  2. 不被浮动元素覆盖
  3. 阻止父子元素之间的margin传递，解决margin-top和margin-bottom的bug

触发BFC的条件：

  1. float的值不为none
  2. position的值不为relative和static
  3. overflow的值不为visible
  4. display的值为table-cell、table-caption和inline-block之一

display:none和visibility:hidden的区别：

display:none是元素彻底消失了，不仅看不见而且不占位置

visibility:hidden是元素看不见了，但是位置还在
