使用hexo框架搭建博客的步骤

发表于 2018-05-23

使用Hexo搭建博客的步骤：

创建基于hexo的项目

  1. npm install hexo-cli -g 安装
  2. hexo init blog（项目名称） 初始化项目
  3. cd blog 进入项目
  4. npm install 下载模块
  5. hexo server

配置主题：NexT主题安装步骤

  1. 在blog项目中 git clone https://github.com/iissnan/hexo-theme-next themes/next
  2. 启用主题：打开站点配置文件_config.yml，找到theme字段，并将其值改为next
  3. 验证主题是否正确启用，先用hexo.clean来清缓存，再用 hexo s –debug 启动

主题设定：选择scheme

  1. 在主题配置文件 _config.yml 中找到 scheme 字段并设置（Muse，Mist，Pisces），用哪个主题就把scheme前的#去掉

设置语言：

  1. 打开站点的配置文件 _config.yml 找到 language 字段，设置成 zh-Hans（简体中文）

设置菜单：

  1. 编辑主题配置文件 _config.yml 找到 menu 字段，需要哪个把哪个前面的#去掉

部署到github：

  1. npm install hexo-deployer-git –save

  2. 打开站点的配置文件 _config.yml 找到 deploy 字段，修改

    deploy:
    type: git
    repo: https://github.com/cxfengmm/cxfengmm.github.io
    branch: master

  3. 部署 hexo g -d
