vue-cli

安装步骤

  1. 全局安装vue-cli：npm install –global vue-cli
  2. 创建一个基于webpack模板的新项目：vue init webpack my-project
  3. 进入项目：cd my-project
  4. 安装依赖：npm install
  5. 运行：npm run dev
  6. 打包：npm run build 生成dist文件夹

主要文件作用

  1. index.html——展示页面
  2. index.js——引入一些配置文件 （import from 同级目录用./，@就是../）
  3. App.vue——根组件，展示页面得结构，引样式表用@import ‘目录’
  4. components——子组件 export default{}
  5. 所有得静态资源全部放在static文件夹下

vue-router

  1. <router-link :to="{name:home,path:'/home',params:{p:1},query:{q:2}}">

    参数：name是跳转到的路由的名称，path是跳转到的路由的路径，params和query是传递的参数；

  2. router-link 默认渲染成一个 a 标签，可以通过 tag 属性指定标签；

  3. 路由组件想在哪渲染就在哪写 router-view 标签；可以在 router-view 标签外面包一个 keep-alive 标签，这样就把渲染过的组件缓存起来了；

  4. 可以使用 watch 来监测路由变化，只要路由变化了就会调用；

  5. 可以在任何组件内通过 this.$router 访问路由器，也可以通过 this.$route 访问当前路由；

vue请求数据——axios

  1. 下载：npm install axios

  2. 引用：import axios from axios

  3. axios的API：

    axios({method:'get/post',url:'./static/resource/get.php',data:{id:1}}).then(function(response{ console.log(response)}).catch(function(error){console.log(error)}) ；

  4. 注意：当 method 是 post 时，可能会出现404错误，是data的数据格式有问题，需要用 URLSearchParams 转换一下；

    var params = new URLSearchParams(); params.append('id','1');

  5. 注意：axios 和 URLSearchParams 不兼容 ie ，需要下载 npm install es6-promise --save 和 npm install url-search-params-polyfill --save 然后再在 main.js 文件中引入import promise from 'es6-promise';promise.polyfill(); import 'url-search-params-polyfill';

数据传递

  1. 父组件向子组件传递数据：通过props，在子组件里写 props:['num']或者props:{num:{}}；在父组件用到子组件的地方写:子组件props的名称（num）="父组件要传递的数据"；父子组件的数据名称不能一样而且大小写会被解析成一样;
  2. 子组件向父组件传递数据：通过事件；
  3. 组件与组件之间的数据传递：中央数据总线或者 vuex
